1 - Utilização da lib tesseract para leitura de caracteres de imagens 
2 - Necessária a presença do tesseract instalado no PC que o servidor estiver rodando
3 - Utilizado o postman como teste para enviar a imagem 
4 - Deve ser enviado a imagem dentro do formato JSON informando sua extensão ex: 
{
    "imagem": "Etiqueta2.JPG"
}
5 - O servidor roda por padrão na porta 5000 
6 - A imagem deve estar no mesmo nível de diretório do servidor
